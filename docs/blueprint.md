# **App Name**: PAW (Power & Water)

## Core Features:

- Record Power: Record power readings from a meter.
- Record Water: Record water readings from a meter.
- Rate Calculation: Calculate individual rates based on the 'mother' bill or reading. The application will act as a tool to distribute the 'mother' bill amount across tenants.
- Theme Switcher: Switch between fire/dark and ocean color schemes via a toggle in the settings panel.
- Data Visualization: Visually represent billing data using charts or graphs.

## Style Guidelines:

- Initial Color Palette (Fire + Dark): Primary color: Deep, saturated red (#D43A3A), evoking fire. This color has the potential to signal a feeling of strength and activity.
- Initial Color Palette (Fire + Dark): Background color: Dark gray (#2B3035) to complement the theme with darkness; a low saturation hue from the same color family as the red creates harmony.
- Initial Color Palette (Fire + Dark): Accent color: A slightly brighter, orange color (#E58E26). It's saturation and lightness levels ensure good contrast. Its placement close to red on the wheel will lend it a fiery association.
- Alternative Color Palette (Ocean): Primary color: Calming ocean blue (#3498DB) is employed, to instill a mood of peace and stability.
- Alternative Color Palette (Ocean): Background color: Very light, desaturated blue (#EBF5FB), of the same hue as the primary, establishes an airy background.
- Alternative Color Palette (Ocean): Accent color: A slightly different shade of blue (#2980B9) offers depth without clashing.
- Clean and modern typography to ensure readability.
- Simple layout with clear sections for readings and billing.
